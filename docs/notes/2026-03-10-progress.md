# 🔌 用 OpenClaw 连接 MCP 服务，自然语言查学生数据

## Student Analytics 开发日记 第2期

上次分享了项目启动和数据模型设计，这次系统已经部署上线了！今天来聊聊怎么用 OpenClaw 连上我们的 MCP 分析服务，体验自然语言查数据的快感~

---

### 🚀 系统已部署上线

经过一系列架构升级和语义模型完善，Student Analytics 已经部署到云服务器：

- 后端服务运行稳定，MySQL + MongoDB 双数据库就位
- 构建了 16 个 TM 表模型 + 16 个 QM 查询模型，覆盖完整的学生数据分析场景
- MCP 协议接口已开放，支持任何兼容 MCP 的客户端接入

---

### 🔌 OpenClaw 一句话接入

打开 OpenClaw，直接跟它说：

> 「帮我把 http://60.204.247.236:8098/mcp/analyst/rpc 这个 MCP 服务配置一下」

OpenClaw 会自动帮你配好，然后就可以直接开始查数据了！

**可以问什么？**

- 「帮我查一下高三一班的数学平均分」
- 「最近一次期末考试，哪个班级的语文成绩最好？」
- 「张三同学的各科成绩趋势怎么样？」
- 「本学期出勤率低于 90% 的学生有哪些？」

语义层已覆盖：学生、班级、年级、科目、考试、成绩、出勤、教师任课等维度。

---

### 📊 语义模型全景

| 类型 | 模型数 | 覆盖范围 |
|---|---|---|
| 维度模型（TM） | 16 个 | 学生、班级、年级、科目、考试、学期、日期、教师等 |
| 查询模型（QM） | 16 个 | 全部维度 + 成绩分析、出勤统计、学习建议、学生画像 |
| MCP Tools | 7 个 | 数据查询、模型描述、元数据获取、导出等 |
