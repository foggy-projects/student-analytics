# 🎯 AI 学生分析系统：自然语言录数据，上线了！

## Student Analytics 开发日记 第3期

上期实现了 MCP 查询服务，这期搞定了反向操作 —— **MCP 数据录入服务**。现在不仅能用自然语言查数据，还能用自然语言录成绩、记考勤、加学生。批次追踪 + 一键回撤，老师录错了也不怕。

---

### 🔧 MCP 录入服务 /mcp/input/rpc

与只读的 `/mcp/analyst/rpc` 完全隔离，新开了一个写入端点。实现了 8 个 MCP Tools：

| Tool | 功能 | 写入类型 |
|------|------|---------|
| `score.create` | 录入单条成绩 | INSERT |
| `score.update` | 修改成绩 | UPDATE + SNAPSHOT |
| `score.batch_create` | 批量录入成绩 | BATCH INSERT |
| `exam.create` | 创建考试 | INSERT |
| `student.create` | 新增学生 | INSERT |
| `attendance.record` | 批量考勤记录 | BATCH INSERT |
| `batch.rollback` | 按批次回撤 | DELETE / RESTORE |
| `batch.list` | 查询操作历史 | SELECT |

核心设计原则：**MCP 只做精准写入，不猜不推断**。模糊匹配、图片识别这些「脏活」全交给 AI 客户端。

---

### 🛡️ 批次追踪与回撤机制

这是本期的亮点设计。每次写入自动分配 `batch_id`，记录到 `sys_batch_log`：

- 成绩录入 → 返回批次号，随时可回撤整批
- 成绩修改 → 自动快照原始值到 `sys_update_snapshot`，回撤时精确还原
- 学生创建回撤 → 自动回退班级人数
- 考试创建回撤 → 检查是否有关联成绩，有则拒绝（安全）

---

### 🧠 智能辅助模块

- **ScoreLevelCalculator**：成绩等级自动计算（A/B/C/D），基于科目的及格线和优秀线配置
- **ClassNameMatcher**：班级名称模糊匹配，「七年级一班」「初一1班」「初一（1）班」都能识别
