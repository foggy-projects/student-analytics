# 🎯 AI学生分析系统开发日记 Day 1

## 项目正式启动啦！

最近在做一个 AI 驱动的学生数据分析系统，今天来汇报下进度~

---

### 📐 架构设计 已完成

技术栈选型敲定：
- 后端：Spring Boot 3.4.5 + Java 17
- 前端：Vue 3 + Vite + TypeScript
- 数据库：MySQL
- AI 能力：Foggy Dataset Model 语义引擎

整体是一个**单校部署**的轻量级分析系统，目标是让老师用自然语言就能查数据、看分析。

---

### 🗄️ 数据模型 已完成

一期 MVP 设计了 **16 张表**，覆盖 5 大业务域：

| 域 | 表数量 | 说明 |
|---|---|---|
| 权限组织 | 3 张 | 用户、教师、任课关系 |
| 时间 | 2 张 | 学期、日期维度 |
| 学生班级 | 3 张 | 年级、班级、学生 |
| 科目考试 | 4 张 | 科目、考试、知识点、闭包 |
| 成绩分析 | 4 张 | 成绩、画像、学习建议、考勤 |

亮点设计：
- 排名不存表，用 QM 窗口函数实时算
- 知识点用闭包表支持多层级聚合
- AI 画像表存 JSON，灵活扩展

数据库 DDL 已经写好，随时可以建库。
